# Bike Purchase Behavior Analysis Dashboard

## Project Overview

This project is an interactive Excel dashboard built to analyze customer bike purchasing behavior using demographic and commute-related data.

The dashboard helps identify trends between customer attributes such as age, gender, income, education, region, and commute distance with bike purchase decisions.

---

## Problem Statement

Businesses often struggle to understand which customer segments are more likely to purchase bikes.

The objective of this project was to analyze customer demographic and lifestyle data to answer questions such as:

- Which age group purchases bikes the most?
- Does income influence bike purchases?
- How does commute distance affect buying behavior?
- Which regions and education levels show higher purchase activity?
- Are there noticeable differences between male and female customers?

The goal was to transform raw customer data into meaningful business insights through an interactive reporting dashboard.

---

## Dashboard Features

- Interactive slicers for:
  - Marital Status
  - Region
  - Education

- Visual analysis of:
  - Average Income by Gender
  - Customer Age Brackets
  - Customer Commute Distance
  - Bike Purchase Trends

- Dynamic filtering using Pivot Tables and Pivot Charts

---

## Tools & Techniques Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization

---

## Key Insights

- Young adults showed the highest bike purchase count among age groups.
- Customers with shorter commute distances were more likely to purchase bikes.
- Male customers had a slightly higher average income compared to female customers.
- Regional and educational segmentation helped identify customer buying patterns.
